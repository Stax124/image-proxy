# Readme

This directory contains the libjxl deb files that are used in the GitHub Actions workflow for testing.
As of the time of writing, the required version of libjxl is 0.11.1, but only 0.7.0 is available in the Ubuntu repositories, so we need to build it ourselves and include the deb files here.

For more information on how to build libjxl, see the [libjxl repository](https://github.com/libjxl/libjxl)
Files taken from https://github.com/libjxl/libjxl/releases/tag/v0.11.1
